三月的個人回顧

一個半月前(1/31)有說想要每個月都要做KPT。結果根本就超過一個月了，今天再不寫大概就要變成雙月刊了ODO，大致列一下這段時間內做了什麼事情。

  • 跟朋友去了一趟富士山（3/2~3/4三天，雖然都是下雨完全看不到山的狀態），之後應該會寫一篇遊記，再看什麼時候可以生出來，現在大概還有三篇左右的草稿還沒完成。
  • 寫了二篇參加conf的心得文。（DroidKaigi 2019 現場，DeNA TechCon 2019）
  • 當初沒列，但其實正在轉職活動了，這個月有空的時間幾乎都是花在這個相關的準備上。之後應該也會寫一些在這段準備時間裡學習到的事情。

Keep

  • 薪水
  • 一年國內旅行2~3次（二天以上）
  • 不加班
  • 寫文章（新）
    • 以目前的狀況來說還沒有出現不知道要寫什麼的情況，就只是需要找時間把想法變成文章而己。

Problem

  • 日文沒什麼長足的進步
  • 工作最近有點變動，己經不滿足當初的目標 → 找到理想的工作
  • 因為轉職的關係多了很多需要做的事情，影響到時間資源的使用（新）

Try

  • 努力準備轉職活動
