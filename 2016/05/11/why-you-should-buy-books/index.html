為什麼買書

重點放前面，買書是為了吸收書本的資訊。

不會看三次以上的書不買

對我而言，書本是俱有重量也很佔空間的東西。在還居無定所需要搬家的狀況下，它會是很大的負擔，不想給自已找麻煩。除了工具書、多次閱讀可以帶來不同啟發的類型以外，很多時候書本只是讀完就被放置在房間的角落而已，租借或是二手的方式會比較適合。

下面是我自已實際有買也推薦看的書：

  • ReWork （工作、創業）
  • Clean Code 無遐的程式碼 （程式）

我會再放上閱讀後所得到的心得，

如果你也有收藏這種類型的書，歡迎留言推薦或討論。
